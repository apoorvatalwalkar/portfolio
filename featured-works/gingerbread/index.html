Gingerbread Workshop

A Hand-Drawn Animation

Context Results
OVERVIEW

Context

Process

For my Foundation Media final project, I wanted to try and make a 2D animation entirely by hand. As this project was close to the start of December, I decided to make a somewhat wintery animation featuring gingerbread cookies as moving characters. After storyboarding and sketching, I rendered every frame in color on Adobe Fresco.

Takeaways

This project was a really fun introduction to animation. Before this I hadn't made any 2D animations by hand. I think it was a really cool starting off point to gauge my current experience and see where I can improve.
