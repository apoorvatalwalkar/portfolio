Room Model

A Maya Exploration

Context Building Furniture Adding Lights Results
OVERVIEW

Context

Process

To start learning Maya I decided to model my own room. Before this I had no experience with Maya or any other modeling software.

Timeline

Self Project • Summer 2025 • 3 weeks

Go to Results
FEATURE 1

Building Furniture

The first step of this project was building furniture. I learned how to build complicated pieces from the basic poly models. After creating the pieces, I added color to each piece to make them look realistic.

FEATURE 2

Adding Lights

After creating and coloring all the pieces I added different lighting to simulate the different lights in my room. This included a skylight outside the window, a candle warmer lamp on the nightstand, a desk lamp, and lights on the computer and monitor.

FINAL

Results

Takeaways

I think this was a really useful project for my initial exploration into modeling. By modeling my room, while I was at home, I was able to inspect the elements I was modeling in real life as I was creating them.
